Weatherstation
==============

About
-----

This weather station software consists of two parts: Data collection and data visualization.

Data is collected from a serial port and put into an Sqlite database.
Data visualization is handled by jqPlot.


Requirements
------------

- Python 2.x
- Web server with support for cgi (Apache/Lighttpd/Python)
